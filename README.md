### App Tarefa React JS
### Criação do projeto
    * Vite create project
    ```
    npm create vite@latest
    ```

    * Vite instalacão de dependencias
    ```
    npm install
    ```

    * Vite Rodando o projeto
    ```
    npm run dev
    ```
### Dependências

    ```
    npm i react-icons
    ```
